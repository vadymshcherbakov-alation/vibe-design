Alert

An Alert is an inline status message — it tells the user something about the region of the view they're looking at. Severity (success, info, warning, error) carries the icon and colour that match. Alation uses plain MUI <Alert> themed with custom @alation/icons-neo icons, the outlined default variant, theme.spacing(2) padding, and body1 / subtitle1 typography — so every Alert reads the same without any sx at the call site. Reach for an Alert when feedback belongs next to the content it describes; use a Dialog if the message must block work.
