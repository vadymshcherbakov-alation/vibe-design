Content Switcher

A Content Switcher is a compact segmented control — a row of mutually-exclusive options sitting inside a grey track, where the selected option lifts as a white pill. It captures which slice of the same content the user is looking at right now — Grid / List, All / Mine / Shared, Daily / Weekly / Monthly — and swaps a small content block in place without navigating away.

Content Switcher, Tabs, and the app shell sit at three different levels of the navigation hierarchy: Tabs split a single page into 2–~7 heavy panels; the App bar / Wayfinder / Top Nav carry top-level destinations; Content Switcher (and Radio) handle the smallest grain — a compact filter pill above a table, a view-mode toggle in a toolbar, a 2–5-option choice inside a form. Pick the level that matches the weight of the content being switched.

Named styles — selection state

Each option is one of two named styles. The lift to a white pill is the selected indicator — there is no separate dot or underline.

Unselected — transparent option in the track; label text.secondary
Selected — option is grey.50 with soft shadow; label text.primary at fontWeightBold

Sizes

Two sizes from the Figma library — Medium (default) and Small. Pick Small only when vertical space is tight (dense toolbar / card header).

Medium (default) — size="medium"

Small — size="small"

Variants — view-mode toggle

Binary opposites where both labels need to stay visible — Grid / List, Compact / Comfortable. When the other label can be implied (Active / Inactive feature flag), reach for a Switch instead.

Variants — filter pill above a data-table view

Single-axis filter above a list — All / Mine / Shared. Persist the active value in a URL query param (?view=mine) so the view is deep-linkable, the same persistence rule Tabs follows.

Variants — inside a Form Field

A 2–5-option form choice where the horizontal segmented look fits better than a Radio column. Wrap in a Form Field when you need a label, helper text, or validation — the inline aria-label alone is enough when none of those apply.

Sync cadence*

Frequency at which the source is re-indexed. Daily is recommended for active sources.

States

Idle (selected / unselected) · Hover · Focus-visible · Disabled (group + single option).

Default · Hover · Focus-visible

Disabled — entire group

Pair with a tooltip on the wrapping element explaining why — e.g. "Connect a source to enable view modes."

Disabled — single option

Pair with a tooltip on the disabled option — e.g. "List view requires data to load first."

Anatomy

Track / container — the <ToggleButtonGroup> · grey-300 pill, 0.5 padding, radius 6
Option — one <ToggleButton> · transparent when unselected, grey.50 pill with soft shadow when selected
Label — text inside the option · body1 (Medium) or body0 (Large) · sentence case, ≤ ~12 chars
Selected indicator — the white pill + soft shadow + fontWeightBold swap (no separate underline / dot)
Group label — external · Typography line or Form Field's <FormLabel> · aria-label mirrors that text
