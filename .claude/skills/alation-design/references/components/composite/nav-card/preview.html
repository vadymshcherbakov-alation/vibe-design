Nav Card

A Nav card is a clickable tile that represents one navigable entity — a data source, an agent, a dashboard, a catalog section. It communicates "this is an object you can open" by pairing a tinted circular icon with a short title and a one-line description. Reach for a Nav card when the user is choosing between a small set of entities (roughly 3–8) and each one deserves a distinct visual identity; for longer lists, use a Table instead.

Classification

  • Type: Composite component — composition pattern, not a shared wrapper
  • MUI base: Card (via the Card Wrapper primitive)
  • Figma: Nav card — to be linked
  • Code: There is no shared <NavCard> component in @alation/alation-ui. Production composes <Card variant="outlined"> + a tinted icon <Box> + <Typography> directly at the call site. This reference documents that composition contract so every tile looks the same across pages.
