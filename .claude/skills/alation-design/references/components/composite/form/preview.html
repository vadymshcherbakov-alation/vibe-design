Form

A vertical stack of labeled fields for creating or editing an entity. Canonical shape: a Stack spacing={3} of Form Field wrappers, each containing a TextField (text, multiline, or select) or an Autocomplete for user/entity search.

Classification

  • Type: Composite component — composition pattern, not a shared wrapper
  • MUI base: none (composition of Stack + FormControl / InputLabel / TextField / Autocomplete)
  • Figma: Form — to be linked
  • Code: There is no shared <Form> component in @alation/alation-ui. Production forms compose <Stack> + the Form Field pattern (form-field.md) directly at the call site. This reference documents that composition.
