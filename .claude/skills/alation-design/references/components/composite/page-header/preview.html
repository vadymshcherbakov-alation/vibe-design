Page Header

The page header sits at the top of every page inside the white main area. It owns a consistent title region (the only mandatory element), wraps it in standard paddings, and optionally hosts a stack of add-ons: a right-side actions group, a Breadcrumb above the title, a Label Chip next to the title, and Tabs below it. Each add-on is an independent enable/disable axis — they layer on top of the title rather than producing a fixed list of variants.

Classification

  • Type: Composite component — composition pattern, not a shared wrapper
  • MUI base: none (composes / Typography / Button / IconButton / MUI Tabs / Breadcrumbs / Chip)
  • Figma: Page Header · NEO 2.1
  • Code: There is **no shared generic component** in @alation/alation-ui. Production assembles page headers ad-hoc at the call site and uses domain-specific wrappers (ObjectHeaderActions, TabHeaderWithNumber) for repeated domains. Prototype code in @repo/ui exposes PageHeaderSection as a layout scaffold — it is prototype-only and not part of the design system contract.

Anatomy

Title is the only mandatory element. Every other element is an independent enable / disable add-on.
Element Status Position Notes
Wrapper (paddings + border) Required Outer container pt: 3 · px: 3 · pb: 2.5; bottom 1px divider (suppressed when Tabs are enabled).
Title Required Title row Typography variant="h1", sentence case.
Actions group Optional Title row, right-aligned Stack of Grey outlined → Blue secondary → Primary → ⋮ IconButton. At most one Primary.
Breadcrumb Optional Above the title row (mb: 1) Trail shape (2+ levels) or Back-to-parent shape (exactly one level).
Label Chip Optional Inside title group, after the title (gap: 1.5) Subtle Label Chip — variant="filledLight", size="xsmall". One concept only.
Tabs Optional Below the header section Header's bottom border is suppressed; the border moves to the wrapper around <Tabs>.
