Form Field

A labeled input unit — the smallest complete block of a form. Pairs one label (optionally with an info tooltip) with one input surface and one helper/error line, so every field looks, reads, and validates the same regardless of what the user is entering.

Classification

  • Type: Composite component — **composition pattern, not a shared wrapper**
  • MUI base: FormControl
  • Figma: Form Field · NEO 2.1
  • Code: @alation/fabric-theme-morpheus — src/lib/ (Label + HelperText overrides + input-surface overrides: TextField / Select / Autocomplete / Radio / DatePicker). There is no shared component in @alation/alation-ui — production composes FormControl + InputLabel + one input + FormHelperText directly at the call site.
